## Цель работы

Целью данной лабораторной работы было изучение основ обработки цифровых изображений

## Что было проделано в рамках подготовки к работе

В рамках подготовки к данной лабораторной работе было выполнено следующее:

* Изучены поэлементные операции, методы на основе построения гистограммы изображения;
* Подобраны изображения, которые наиболее ярко будут показывать действие алгоритмов, которые необходимо реализовать

## Написание программы

Реализованное приложение является web-приложением. На этапе написания приложения был сделан выбор в пользу классической
связки html5/css/js для совместимости даже с уже устаревшими браузерами.
Корректность работы приложения была протестирована вручную в браузере Firefox.

## Описание приложения

Приложение позволяет пользователю загружать, а затем интерактивно изменять изображение при помощи следующих методов
обработки и сжатия изображений: Линейное контрастирование, Эквализация гистограммы RGB, Эквализация гистограммы HSV,
Размытие по Гауссу, Однородный усредняющий фильтр.

## Выводы

В ходе выполнения лабораторной работы были изучены и применены основные методы обработки цифровых
изображений. Разработка интерфейса на базе HTML5/CSS/JavaScript обеспечила широкую совместимость приложения и
возможность его
использования в различных браузерах без необходимости установки дополнительного программного обеспечения. Проведённое
тестирование подтвердило корректность работы реализованных функций.

Полученные результаты показали, что выбранные методы обработки позволяют эффективно изменять характеристики изображений,
улучшать их визуальное восприятие и выполнять базовые операции по повышению качества. Лабораторная работа способствовала
закреплению теоретических знаний и формированию практических навыков разработки и тестирования инструментов для цифровой
обработки изображений.
