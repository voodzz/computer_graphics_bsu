## Описание

Приложение, принимающее на вход координаты начала и конца линий, которые необходимо изобразить пошаговым методов,
методом ЦДА и методом Брезенхема для линий или окружностей. Выдаёт изображение построенных линий.

## Цель работы

Разработать приложение/веб-приложение для иллюстрации работы базовых растровых алгоритмов, таких как пошаговый
алгоритм, ЦДА и алгоритм Брезенхема.

## Задачи работы

* Реализовать пошаговый алгоритм для рисования отрезков.
* Реализовать алгоритм ЦДА для рисования отрезков.
* Разработать алгоритм Брезенхема для построения отрезков.
* Создать пользовательский интерфейс, включающий масштабирование, вывод системы координат, осей, линий сетки и их
  подписей.
* Указать временные характеристики реализованных
  алгоритмов

## Использованные средства разработки

* HTML5
* JavaScript
* CSS

## Ход работы

* Разработаны и реализованы алгоритмы для построения отрезков.
* Создан пользовательский интерфейс приложения, обеспечивающий удобное масштабирование и отображение системы координат,
  осей и сетки.
* Оформлен краткий отчет с временными характеристиками работы реализованных алгоритмов.
